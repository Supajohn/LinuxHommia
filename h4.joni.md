4.12.1 Size and Complexity of Some DSLs
DSL:t (Domain Specific Language) voivat sisältää hyvin suuren määrän ominaisuuksia, mutta käytännössä vain osa niistä on aktiivisessa käytössä
Monimutkaisuus kasvaa nopeasti ominaisuuksien määrän mukana

4.12.2 Use of DSL Functions in Case Configuration
Vaikka DSL tarjoaa paljon toimintoja, käyttäjät hyödyntävät vain pientä ydinosaa
Usein käytetään perusoperaatioita kuten:
pakettien asennus
tiedostojen hallinta
palveluiden käynnistys

4.12.3.1 Dependencies Between Main Functions
Konfiguraatioissa eri toiminnot ovat riippuvaisia toisistaan
Esimerkiksi:
paketti täytyy asentaa ennen kuin palvelu voidaan käynnistää
asetustiedosto pitää olla olemassa ennen palvelun reloadia
Näistä muodostuu perusrakennuspalikat (building blocks)
